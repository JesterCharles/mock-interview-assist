## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → /api/coding/submit | Untrusted code payload; server must rate-limit, auth-gate, language-allowlist |
| trusted markdown (trainer-authored) → DOM | Rendered via `dangerouslySetInnerHTML`; source is trusted but future user-authoring would change this |
| Monaco bundle → client | Must be lazy-loaded; no secrets in Monaco workers |
| challenge.starters → editor | Server-sourced code; React auto-escapes in `value` prop (no HTML interp); Monaco treats as plain text |
