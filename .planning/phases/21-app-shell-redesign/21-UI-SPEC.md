## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label gap in sidebar items, badge internal padding |
| sm | 8px | Sidebar item internal vertical padding, topbar item gap |
| md | 16px | Topbar horizontal item spacing, sidebar group padding, section content padding |
| lg | 24px | Sidebar group label top margin, topbar height padding |
| xl | 32px | Content area left margin from sidebar edge |
| 2xl | 48px | Major section breaks inside content |
| 3xl | 64px | Page-level top padding below topbar |

Exceptions:
- Touch targets (hamburger button, avatar button, sidebar toggle): minimum 44px height/width per `btn-accent-flat` / `btn-secondary-flat` pattern in `globals.css`
- Topbar height: 56px (14px × 4 — fits logo + nav items without crowding)
- Sidebar expanded width: 200px (DESIGN.md §Layout)
- Sidebar collapsed width: 48px (icon-only; aligns with 16px icon + 16px total padding)

Source: DESIGN.md §Spacing, globals.css button system, CONTEXT.md D-13

## Color

| Role | Token | Hex (light) | Usage |
|------|-------|-------------|-------|
| Dominant (60%) | `--bg` | `#F5F0E8` | Page background behind sidebar and content |
| Secondary (30%) | `--surface-muted` | `#F0EBE2` | Sidebar background, topbar background, dropdown backgrounds |
| Accent (10%) | `--accent` | `#C85A2E` | Active topbar nav item indicator, active sidebar item left border/text, primary CTA buttons |
| Destructive | `--danger` | `#B83B2E` | Sign Out button in avatar menu (text color only, no bg fill) |

Accent reserved for:
1. Active/selected topbar navigation item (underline or left-border indicator)
2. Active/selected sidebar item (left 2px border + text color)
3. Primary action buttons (`.btn-accent-flat`)

NOT used for: inactive nav items, hover states (use `--highlight`), group labels, borders

Hover states:
- Nav item hover: `background: var(--highlight)` (`#FFF8F0`), border-radius 6px
- Sidebar item hover: same `--highlight` background, 150ms ease-out
- Active item: `--accent` left border (2px) + `--accent` text color

Surfaces:
- Topbar: `background: var(--surface-muted)`, `border-bottom: 1px solid var(--border)`
- Sidebar (expanded): `background: var(--surface-muted)`, `border-right: 1px solid var(--border)`
- Mobile overlay: `background: rgba(0,0,0,0.30)` (matches RESEARCH.md Pattern 5)
- Avatar dropdown: `background: var(--surface)`, `border: 1px solid var(--border)`, `border-radius: 8px`
- Cohort switcher dropdown: same as avatar dropdown

Source: DESIGN.md §Color, globals.css tokens, CONTEXT.md decisions

## Component Inventory

### TopBar.tsx
- Height: 56px, `position: sticky; top: 0; z-index: 30`
- Background: `var(--surface-muted)`, border-bottom: `1px solid var(--border)`
- Left zone: NLM wordmark (Clash Display 600, links to `/trainer`) + hamburger button (mobile only, `md:hidden`)
- Center zone: Section nav links — Dashboard, Interviews, Question Banks, Settings (hidden on mobile — only hamburger shows)
- Right zone: CohortSwitcher (wrapped in `<Suspense fallback={null}>`) + ThemeToggle + AvatarMenu
- Active detection: `usePathname().startsWith(prefix)` — accent left underline or bottom border on active item
- Breakpoint: hamburger replaces center nav at `< 768px`

### SectionSidebar.tsx
- Expanded width: 200px; collapsed width: 48px
- `position: fixed` on desktop (or `sticky` if layout allows scroll)
- Collapse transition: `width` CSS transition, 200ms ease-in-out (DESIGN.md §Motion "short" duration)
- Collapsed state: icons only, no labels, tooltip on hover showing label
- Toggle button: bottom of sidebar, `ChevronLeft` / `ChevronRight` from lucide-react
- Group labels: 11px JetBrains Mono 500, uppercase, hidden when collapsed
- Item active state: 2px `--accent` left border + `--accent` text color
- Item hover: `--highlight` background, 150ms ease-out, border-radius 6px
- localStorage key: `nlm_sidebar_collapsed` (read in `useEffect` only — no SSR flash)

### AvatarMenu (Radix DropdownMenu)
- Trigger: avatar circle (initials from email) + chevron
- Avatar size: 32px circle, `background: var(--surface-muted)`, `border: 1px solid var(--border)`, 13px DM Sans 500 initial
- Items: user email (non-interactive, 14px DM Sans 400 muted), role badge (admin/trainer — 12px, DM Sans 600, `--success` for trainer, `--warning` for admin), separator, Sign Out (16px DM Sans 400, `--danger` text)
- Dropdown position: `side="bottom" align="end"`, `sideOffset: 8`
- Click-away: Radix handles natively

### CohortSwitcher (Radix DropdownMenu)
- Trigger: cohort name or "All Cohorts" + `ChevronDown` icon
- Trigger width: max 160px, text truncated with ellipsis
- Items: "All Cohorts" (top, always present) + one item per cohort showing name + associate count in muted text (e.g. "Cohort A  12")
- Selected item: checkmark (`Check` icon from lucide) in `--accent` color
- localStorage key: `nlm_cohort_id` (fallback on mount, URL param `?cohort=<id>` is source of truth)
- Wrapped in `<Suspense fallback={null}>` in TopBar

### MobileSidebar (Radix Dialog off-canvas)
- Opens from left, `inset-y-0 left-0 w-64`
- Background: `var(--surface)`, `border-right: 1px solid var(--border)`
- Overlay: `fixed inset-0 bg-black/30 z-40`
- Close button: `X` icon, top-right of panel
- Content: same SidebarItem list as desktop, always expanded (no collapse in mobile sheet)
- Trigger: `Menu` icon in TopBar (visible `< 768px` only)

### Placeholder pages (Gap Analysis, Calibration, Reports)
- Layout: centered column, max-width 480px
- Heading: 28px Clash Display 600, `--ink`
- Subtext: 16px DM Sans 400, `--muted`
- Copy: see Copywriting Contract below

Source: CONTEXT.md decisions D-01 through D-15, RESEARCH.md Patterns 1–5, DESIGN.md

## Interaction States

| Surface | State | Visual Treatment |
|---------|-------|-----------------|
| Topbar nav item | Default | 13px DM Sans 500, `--ink` |
| Topbar nav item | Hover | `--highlight` bg, border-radius 6px, 120ms ease-out |
| Topbar nav item | Active (path match) | `--accent` text + 2px bottom border in `--accent` |
| Sidebar item | Default | 13px DM Sans 500, `--ink`, left border transparent |
| Sidebar item | Hover | `--highlight` bg, border-radius 6px, 150ms ease-out |
| Sidebar item | Active | 2px `--accent` left border, `--accent` text |
| Dropdown item | Hover | `--highlight` bg |
| Mobile overlay | Visible | `bg-black/30`, fade-in 150ms ease-out |
| Sidebar collapse | Transition | width 200ms ease-in-out, labels fade out 100ms |
| Sidebar expand | Transition | width 200ms ease-in-out, labels fade in 150ms (delayed 50ms) |

Focus states (keyboard nav):
- All interactive elements: `outline: 2px solid var(--accent); outline-offset: 2px` on `:focus-visible`
- Radix Dialog and DropdownMenu handle focus trap and keyboard nav natively

Source: DESIGN.md §Motion, globals.css button system, CONTEXT.md D-13
