```bash
# Outputs the digest (without `sha256:` prefix) of the latest successful deploy-staging.yml run.
# Used by runbook + prod.tfvars seeding.

REPO="JesterCharles/mock-interview-assist"
```

**promote-staging-digest-to-prod.sh** (bash, `set -euo pipefail`):
```bash
# Copies the staging-validated image to the prod Artifact Registry so Cloud Run prod can pull by digest.
# Usage: ./promote-staging-digest-to-prod.sh <digest-hex>
set -euo pipefail
DIGEST="${1:-$(./fetch-latest-staging-digest.sh)}"
SRC="us-central1-docker.pkg.dev/nlm-staging-493715/nlm-app/nlm-app@sha256:${DIGEST}"
DST="us-central1-docker.pkg.dev/nlm-prod/nlm-app/nlm-app:v1.5.0-rc1"
gcloud artifacts docker tags add "$SRC" "$DST" --quiet
# Verify the same digest landed in prod registry:
PROD_DIGEST=$(gcloud artifacts docker images describe "$DST" --format='value(image_summary.digest)' | sed 's/^sha256://')
[ "$PROD_DIGEST" = "$DIGEST" ] || { echo "digest mismatch after promote: expected $DIGEST got $PROD_DIGEST" >&2; exit 1; }
echo "$DIGEST"
```
