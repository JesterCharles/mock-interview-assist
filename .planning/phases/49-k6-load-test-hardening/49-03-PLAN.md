Attack-mode matrix (5 modes per D-09):
1. `unauth-get` — GET with no Authorization header
2. `unauth-post-empty` — POST with no Authorization, empty body `{}`
3. `unauth-post-fake-id` — POST with no Authorization, body `{ userId: "00000000-0000-0000-0000-000000000000" }`
4. `expired-token` — any method with `Authorization: Bearer <expired JWT fixture>`
5. `wrong-role` — trainer token against `/api/associate/*`, associate token against `/api/trainer/*` (D-09)

Error-body denylist (from phase_specific_constraints — MUST include ALL of these):
```typescript

  /prisma\//,                      // ORM internals
```
