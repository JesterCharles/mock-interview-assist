    - Named export `extrapolateCost(input: { totalRequests: number; totalVcpuSeconds: number; totalGbSeconds: number }): { costPer1kRequests: number; breakdown: { vcpu: number; memory: number; requests: number } }`.

      - `requests_cost = (totalRequests / 1_000_000) * 0.40`

      - `costPer1kRequests = total / (totalRequests / 1000)`
