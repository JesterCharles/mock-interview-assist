Create `src/lib/emailParser.ts` with:

```typescript
export type ChipState = 'valid' | 'invalid' | 'duplicate';

export interface ParsedEmail {
  value: string;        // original email string (trimmed)
  normalized: string;   // lowercased for dedup comparison
  state: ChipState;
}

export interface ChipSummary {
  valid: number;
  invalid: number;
  duplicate: number;
}
```

Function `parseEmails(raw: string): ParsedEmail[]`:
- Split input on `/[\s,]+/`, trim each part, filter out empty strings
- For each part: validate with `/^[^\s@]+@[^\s@]+\.[^\s@]+$/` regex
- Track seen lowercase emails in a `Set<string>` — second occurrence is 'duplicate'
- Return array of ParsedEmail objects

Function `getChipSummary(chips: ParsedEmail[]): ChipSummary`:
- Count chips by state, return `{ valid, invalid, duplicate }`

Function `isOverCap(chips: ParsedEmail[], cap: number): boolean`:
- Return `chips.filter(c => c.state === 'valid').length > cap`

Function `removeChip(chips: ParsedEmail[], index: number): ParsedEmail[]`:
- Remove chip at index, then re-classify remaining chips (a former duplicate may become valid if the original was removed)
- Re-run dedup logic: iterate chips in order, rebuild seen set, reclassify valid/duplicate states (invalid stays invalid)

Write tests FIRST in `src/lib/emailParser.test.ts` covering all behaviors above. Run tests (RED). Then implement. Run tests (GREEN).
