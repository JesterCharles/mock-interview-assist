  <name>Task 1: Implement manifest cache module + tests</name>
  <files>src/lib/githubManifestCache.ts, src/lib/githubManifestCache.test.ts</files>
  <behavior>
    Test cases (Vitest, 6-8 total):
    1. Cold fetch: empty cache → calls fetcher → stores entry → returns {files, lastSyncedAt, etag}.
    2. Warm hit: second call within TTL → does NOT call fetcher → returns same entry.
    3. TTL expired + 304: fetcher called with If-None-Match, returns {status:304, etag:same} → cache entry files UNCHANGED, expiresAt reset, lastSyncedAt updated.
    4. TTL expired + 200: fetcher returns new body+etag → cache replaces files, etag, lastSyncedAt, expiresAt.
    5. Invalidate(scope): removes key; next call refetches.
    6. Invalidate('all'): clears all keys.
    7. Stampede: two concurrent getManifest() calls on cold cache → fetcher invoked exactly once, both promises resolve to identical result.
    8. (optional) Truncated tree: fetcher returns truncated=true → result returned + console.warn called.
  </behavior>
  <action>
    Create `src/lib/githubManifestCache.ts` mirroring `src/lib/pinAttemptLimiter.ts` pattern (module-level state, `__resetAll` test hook, no class):

    - Constant: `export const MANIFEST_TTL_MS = 5 * 60_000;`
    - Key builder: `const cacheKey = (o,r,b) => \`${o}/${r}@${b}\`;`
    - `const store = new Map<string, { files: GitHubFile[]; etag: string; lastSyncedAt: number; expiresAt: number }>();`
    - `const inFlight = new Map<string, Promise<ManifestResult>>();`
    - Injectable fetcher type: `type FetcherFn = (args: {owner,repo,branch,etag?:string}) => Promise<{status:200; files:GitHubFile[]; etag:string; truncated:boolean} | {status:304; etag:string}>;`
    - Default fetcher hits `https://api.github.com/repos/${owner}/${repo}/git/trees/${branch}?recursive=1` with `Authorization: token ${GITHUB_TOKEN}` (read from `process.env.GITHUB_TOKEN`) and `If-None-Match` when etag provided. On 304 return `{status:304, etag}`. On 200 normalize tree entries: filter `type==='blob' && path.endsWith('.md')`, map to `GitHubFile` with synthesized `name = path.split('/').pop()`, `download_url = https://raw.githubusercontent.com/${owner}/${repo}/${branch}/${path}`, preserve `sha`, `size`, `path`, set `type:'file'`. Log `console.warn('[manifest-cache] tree truncated', key)` if `truncated`.
    - `getManifest(owner, repo, branch)`:
        1. Compute key. If `inFlight.has(key)` → return that promise (log `stampede-dedup`).
        2. Look up `store.get(key)`. If entry and `now < expiresAt` → log `hit`, return `{files, lastSyncedAt, etag}`.
        3. Otherwise create promise: if entry exists call fetcher with `etag`; else without. Store in `inFlight`. On settle: `inFlight.delete(key)`.
        4. On 304: update `entry.lastSyncedAt = now; entry.expiresAt = now + MANIFEST_TTL_MS`; log `revalidate-304`.
        5. On 200: replace store entry with new body+etag+timestamps; log `miss` (no prior entry) or `revalidate-200`.
        6. Return `{files, lastSyncedAt, etag}` from updated entry.
    - `invalidate(scope)`:
        - No arg or `'all'` → `const n = store.size; store.clear(); inFlight.clear(); return n;` log `invalidate scope=all n=${n}`.
        - Object scope → delete specific key; return `1` if deleted else `0`.
    - Test hooks: `export function __setFetcher(fn: FetcherFn | null): void` (null restores default); `export function __resetAll(): void { store.clear(); inFlight.clear(); __setFetcher(null); }`.

    Create `src/lib/githubManifestCache.test.ts` with Vitest suite implementing the 8 behavior cases above. Use `beforeEach(__resetAll)`. Inject a mock fetcher via `__setFetcher(vi.fn(...))`. Do NOT hit real fetch. For stampede test, use a fetcher that returns a promise resolved via deferred; assert `fetcher.mock.calls.length === 1`.

    Follow CLAUDE.md — no speculative config, no env-driven TTL.

    ```typescript
    import { NextRequest, NextResponse } from 'next/server';
    import { getCallerIdentity } from '@/lib/identity';
    import { invalidate } from '@/lib/githubManifestCache';

    const OWNER = 'JesterCharles';
    const REPO = 'mock-question-bank';
    const BRANCH = 'main';

    type InvalidateBody =
      | { scope?: 'all' }
      | { scope: { owner: string; repo: string; branch: string } };

    export async function POST(request: NextRequest) {
      const caller = await getCallerIdentity(request);
      if (caller.type !== 'trainer') {
        return NextResponse.json({ error: 'unauthorized' }, { status: 401 });
      }
      let body: InvalidateBody = {};
      try { body = await request.json(); } catch { /* empty body ok */ }

      let cleared: number;
      if (body.scope === 'all') {
        cleared = invalidate('all');
      } else if (body.scope && typeof body.scope === 'object') {
        cleared = invalidate(body.scope);
      } else {
        // Omit → current default repo/branch (per CONTEXT D7)
        cleared = invalidate({ owner: OWNER, repo: REPO, branch: BRANCH });
      }
      console.log('[manifest-cache] invalidate cleared=' + cleared);
      return NextResponse.json({ cleared });
    }
    ```
    Contract matches CONTEXT D7: `scope: 'all'` → clear all keys; `scope: {owner,repo,branch}` → clear one key; omit → clear the default repo/branch key. `invalidate()` module function already supports both shapes (Task 1).
