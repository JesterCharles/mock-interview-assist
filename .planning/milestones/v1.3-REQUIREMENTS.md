## Notes on Changed Requirements

- **VIZ-03 (Per-skill trend chart)** — Cut formally in Phase 34. The SkillRadar Before/Now polygon overlay supersedes the per-skill LineChart as the canonical trajectory visualization. DESIGN.md §Trajectory Language documents the cut rationale.
- **VIZ-06 (Dashboard skill filter)** — Reconciled in Phase 34 to match 2-component reality (SkillCardList ↔ SkillRadar). Originally specced 3-component sync; SkillTrendChart was removed with VIZ-03.
- **VIZ-07 (Real historical snapshots)** — Introduced in Phase 34 to replace the `0.85 * current` synthetic "Est. prior" polygon. `GapScore.prevWeightedScore` captured inline at upsert.
- **SHELL-01 (Associate nav)** — Reconciled from 2-item to 3-item nav (Dashboard, Interviews, Curriculum) to match UI-SPEC D-04. Phase 35 wired the Settings accordion to complete SHELL-32-01 on the associate side.
- **SIGNIN-02 (First-login gate)** — Phase 28 closed the associate path only; Phase 33 closed the trainer path by reordering the exchange route check + adding an inline SignInTabs client gate.

## Out of Scope (Not Re-Audited)

| Feature | Reason |
|---------|--------|
| OAuth providers (Google, GitHub) | Adds config surface + user confusion about auth method; defer to v1.4+ |
| Leaderboards / cohort comparison | Demotivates lower performers; self-comparison only |
| Curriculum editing by associates | Curriculum is trainer-owned; creates conflict resolution complexity |
| Completion checkboxes on curriculum | Creates anxiety about missed weeks; curriculum is informational |
| Multiple simultaneous recommendations | Decision fatigue; single focus area is more actionable |
| Auto-start recommended mock | Removes associate agency; show CTA instead |
